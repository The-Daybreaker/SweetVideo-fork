# 流心视频
## 简介
- 一款运行在 HarmonyOS Next 上的精致、简约的本地视频播放器，使用 ArkTS 语言开发，具有美观的设计和优雅的动画。
- 基于流心视频 https://gitee.com/lqsxy/sweetvideo/tree/master 
- 本应用根据原作者授权并基于 MIT 协议二次开发而来。
